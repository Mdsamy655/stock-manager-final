1. Detect Bangladeshi mobile numbers (11 digits starting with 01).

2. Support both:
   
   - English digits (016XXXXXXXX)
   - Bangla digits (০১৬XXXXXXXX)

3. Automatically convert Bangla digits to English:
   ০→0
   ১→1
   ২→2
   ৩→3
   ৪→4
   ৫→5
   ৬→6
   ৭→7
   ৮→8
   ৯→9

6. Remove detected phone from input before further parsing.

Name and Address must remain in original language.

STEP 4: NAME DETECTION (BANGLA + ENGLISH)

AI must understand name can be:

Single word:

- শাহরিয়ার
- Rahim

Two words:

- শাহরিয়ার আহমেদ
- Rahim Uddin

Three words:

- মোঃ শাহরিয়ার আহমেদ
- Md Shahriar Ahmed

Name Detection Rules:

1. Name contains only letters (Bangla or English).

2. Name does NOT contain digits.

3. Name does NOT contain address keywords.

4. Common Bangla prefixes increase name probability:
   
   - মোঃ
   - মো:
   - Md
   - Md.
   - Mohammad
   - Muhammad
   - Sheikh
   - Sk

5. If only one word exists before phone:
   → Treat it as Name, NOT Address.

6. If no phone exists:
   → First 1–2 words likely Name.
   → Remaining likely Address.

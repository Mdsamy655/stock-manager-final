SECTION 3: BANGLA NAME DETECTION LOGIC

Bangla Name characteristics:

1. Usually 1–3 words.
2. Contains no numbers.
3. Does not contain address keywords.
4. Usually short (less than 30 characters).
5. Appears first or near phone.

If first short line has no numbers
→ Strong Name candidate.

If input format:

আকাশ ০১৭৯০৫৭৪৫৫৮ বরিশাল সদর

→ First word = Name
→ Middle number = Phone
→ Rest = Address
